                    Private Maxim (DevExpress Support) Private Maxim (DevExpress Support) 4 years ago

                      Hi Michael,
                      Please update the code in the following way:

                      private void button1_Click(object sender, EventArgs e) {  
    var okd = (chartControl2.Diagram as XYDiagram);  

    // Show crosshair  
    ControlCoordinates fffdd = okd.DiagramToPoint("f2", 2);  
    (chartControl2.Diagram as XYDiagram).ShowCrosshair(fffdd);  

    // Show tooltip  
    var seriesPoint = chartControl2.Series[0].Points.Where(p => ((SeriesPoint)p).Argument == "f2" && ((SeriesPoint)p).Values[0] == 2).FirstOrDefault();  
    if (seriesPoint != null) {  
        ChartHitInfo hi = chartControl2.CalcHitInfo(fffdd.Point);  
        var point = chartControl2.PointToScreen(fffdd.Point);  

        ToolTipControlInfo info = new ToolTipControlInfo(seriesPoint, "test") { ToolTipPosition = point };  
        defaultToolTipController1.DefaultController.ShowHint(info);  
    }  
}

                      As you can see, I used the PointToScreen method of the target chart control. Feel free to contact me if you have further questions.
